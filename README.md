# Clube Desportivo Esfera Andebol Masters - Website

Este Projeto consiste na criação de um WEB site para o clube desportivo esfera andebol masters.

O objetivo é aproximar e facilitar a venda de bilhetes de jogos para sócios, aproxiando tanto os socios antigos como os novos

Com o desenvolvimento deste projeto, é possível que os sócios, sócios premium e adeptos do clube possam:
- Comprar bilhetes
- Gerir bilhetes
- Inscrever atletas na formação do clube
- Fazer compras de produtos na loja do clube

Como sócio, também é possível:
- Gerir as suas cotas de sócio
- Fazer a sua própria gestão de gastos relacionados ao clube através da área de sócio

Algumas funcionalidades que ainda não estão disponíveis, como:
- Pagamentos online
- Carrinho de compras na aba de comprar produtos oficiais do clube
- Melhorias da aplicação responsível para telemóvel

Em relação aos pagamentos que ainda não podem ser feitos online, têm de ser feitos na secretaria do pavilhão do clube.
Mas, na compra no site, o utilizador tem o produto/bilhete reservado.

Os utilizadores ainda têm disponível no site uma mapa interativo com uma vista de cada setor do pavilhão antes da compra de um bilhete.

O Site conta ainda com uma página de administrador para facilitar a gestão do site por parte dos responsaveis do mesmo.

Para mais informações, pode ler o manual de utilizador do site que está disponível no meu linkedin que aparece no meu perfil do git hub.

Nota : Este projeto não foi publicado publicamente na internet com um dominio , este projeto destinou-se á minha PAP do curso profissional
